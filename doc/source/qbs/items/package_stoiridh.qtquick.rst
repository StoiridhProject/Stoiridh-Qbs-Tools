Introduction
^^^^^^^^^^^^

The Stoiridh.QtQuick package provides items to create a Qt Quick project.

Import
^^^^^^

So to use this package in your project, you can use the following import statement in your ``.qbs``
file.

.. code-block:: qbs

   import Stoiridh.QtQuick
