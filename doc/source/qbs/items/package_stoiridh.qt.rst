Introduction
^^^^^^^^^^^^

The Stoiridh.Qt package provides items to create a Qt project.

Import
^^^^^^

So to use this package in your project, you can use the following import statement in your ``.qbs``
file.

.. code-block:: qbs

   import Stoiridh.Qt
